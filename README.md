# Ivy Learning Lab

Static website for Ivy Learning Lab — a Dublin-based learning consultancy.

## GitHub Pages Setup

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Under **Source**, select your branch (e.g. `main`) and the root folder `/`
4. Click **Save** — your site will be live at `https://yourusername.github.io/repository-name/`
